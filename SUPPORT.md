# Support

Sapien is a conceptual research architecture. It does not have
a working implementation yet. This means most support questions
will be about the ideas, the architecture, or how to contribute —
not about running code.

---

## Where to Get Help

**For questions about the architecture:**
Open a `[QUESTION]` issue on GitHub.
Include what you read, what confused you, and what you were
trying to understand.

**For discussion and debate:**
Join the Forestritium Discord server.
There is a dedicated Sapien channel.
[discord.gg/Q64wr956](https://discord.gg/Q64wr956)

**For identified flaws or problems:**
Open a `[FLAW]` issue on GitHub.
See CONTRIBUTING.md for the correct format.

**For research questions:**
Open a `[RESEARCH]` issue on GitHub.
Include the paper or work you want to discuss and its relevance
to Sapien.

**For everything else:**
Open a general issue on GitHub with a clear description of
what you need.

---

## What Is Not Supported

- There is no working code to debug yet
- There is no installation process yet
- There are no runtime errors to fix yet

If you are looking for a working implementation to run,
Sapien is not there yet. See the project roadmap in README.md
for current status.

---

## Response Time

This is a solo project maintained by one person alongside
academic studies. Response times may vary. Patience is appreciated.

Priority order for responses:
1. Critical flaw identification
2. Architecture questions
3. Research contributions
4. General discussion
